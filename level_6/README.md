## Module output declarative syntax 

```hcl
#outputs.tf 

output "s3_bucket_name" {
  description = "S3 bucket name."
  value       = module.s3.bucket_name
}

output "s3_bucket_arn" {
  description = "S3 bucket ARN."
  value       = module.s3.bucket_arn
}
```

 `value = module.s3.bucket_name`
